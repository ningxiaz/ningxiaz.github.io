LIVEMANIAExperience live concerts on your couch.

LiveMania is a web app designed for Google TV. It allows users to quickly find live music videos by typing keywords of artists, tracks or albums. It's designed for 10-foot experience so that users can discern the UI elements easily and interact with it via minimal keypad operations.

You can find the live version here and just try it with Chrome on laptop. The github repository is here.

This is an individual project. Before getting my hands on it, I carefully investigated the design guidelines for smart TVs. Then I did storyboarding and wireframes according to the guidelines and some brief user research. Starting with the template provided by Google, I pulled the data from Last.fm using JSON and streamed the right videos from Youtube. There were some technical difficulties of wrapping Youtube videos in HTML5 video element and accessing Youtube from China. Fortunately they were conquered at last.
