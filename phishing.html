PHISHING DETECTIONMake email safer with Neural Networks.

The goal of this project is to apply multilayer feedforward neural networks to phishing email detection and evaluate the effectiveness of this approach. From the statistical analysis, we conclude that NNs with an appropriate number of hidden units can achieve satisfactory accuracy even when the training examples are scarce. Moreover, our feature selection is effective in capturing the characteristics of phishing emails, as most machine learning algorithms can yield reasonable results with it.

This was an individual research project I did at University College Dublin as a summer intern in 2011, where I self-learned the concepts of NN, wrote some Perl scripts to process the dataset and built a very simple NN with Encog Java Core package. Later I took CS229 Machine Learning by Andrew Ng at Stanford and learned more algorithms and evaluation methods that I wished to continue with the project. So I found another partner in Statistics. Together we designed the feature set, reviewed the scripts, wrote Matlab programs to automate experiments, applied other major machine learning algorithms, conducted statistical analysis and created visualizations for the results.
